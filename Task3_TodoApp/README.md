# OIBSIP – DayDone To-Do App
Live demo : https://charan-y16.github.io/OIBSIP/Task3_TodoApp/

## Level 2 Task 3

DayDone is a modern and responsive To-Do List web application developed as part of the Oasis Infobyte Web Development & Designing Internship (OIBSIP). This project helps users manage daily tasks efficiently with features such as task creation, completion tracking, editing, deletion, and progress monitoring.

---

## Project Overview

The DayDone To-Do App is designed to provide a clean and user-friendly interface for organizing tasks. It includes separate sections for pending and completed tasks, real-time statistics, and a progress bar to visually track productivity.

---

## Features

- Add new tasks quickly - Mark tasks as completed or pending - Edit existing tasks - Delete tasks - Separate sections for pending and completed tasks - Real-time statistics for total, pending, and completed tasks - Progress bar showing completion percentage - Responsive design for desktop and mobile devices - Smooth animations and modern UI design - Keyboard support for adding and editing tasks

---

## Technologies Used

- HTML5 – Structure of the application - CSS3 – Styling, responsive design, and animations - JavaScript – Task management logic and interactivity - Google Fonts – Nunito and JetBrains Mono for typography

1. Download or clone the repository. 2. Navigate to the Task3_TodoApp folder. 3. Open the index.html file in any web browser. 4. Add, edit, complete, and manage tasks using the application.

# OIBSIP – VaultIn Authentication System
Live demo : https://charan-y16.github.io/OIBSIP/Task4_AuthenticationSystem/
## Level 2 Task 4

VaultIn is a responsive authentication web application developed as part of the Oasis Infobyte Web Development & Designing Internship (OIBSIP). This project demonstrates a complete front-end authentication interface with registration, login, validation, password strength checking, and a secure dashboard experience.

---

## Project Overview

The VaultIn Authentication System provides a modern and user-friendly interface for creating accounts and signing in. It includes form validation, password visibility toggling, password strength indication, error handling, and a simple dashboard displayed after successful authentication.

---

## Features

- User registration form - User login form - Form validation for name, email, and password fields - Password strength indicator - Password confirmation validation - Password visibility toggle - Error messages for invalid input - Success and error toast notifications - Responsive design for desktop and mobile devices - Simple dashboard page after successful login - Sign out functionality

---

## Technologies Used

- HTML5 – Structure of the application - CSS3 – Styling, responsive design, and UI effects - JavaScript – Authentication logic and form validation - Google Fonts – Nunito and JetBrains Mono for typography

## How to Run the Project

1. Download or clone the repository. 2. Navigate to the Task4_AuthenticationSystem folder. 3. Open the index.html file in any web browser. 4. Create an account, log in, and explore the dashboard interface.
